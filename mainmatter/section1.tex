\section*{Notation}
Standard probability theory notation is adopted. In particular, the following notations are used:
\begin{itemize}
    \item   $X$ for random variables with sample space $\mathcal{X}$  
    \item   $\mathcal{P}(\mathcal{X})$ for the set of all probability measures on the Borel $\sigma$-algebra over $\mathcal{X}$
    \item   $P \in \mathcal{P}(\mathcal{X})$ for probability measures
    \item   $p_X$ for the density function of a probability measure $P$, index might be dropped if notation is clear
    \item   $X \sim p$ for a random variable X whose distribution is described by $p$
\end{itemize}
When discussing neural networks, the symbol $q$ or $q_\theta$ will be used for the model of a true distribution $p$.
